Блокировка пользователей

Заблокировать доступ к OpenVPN для пользователя можно двумя способами:

  • Через список активных подключений
  • Через страницу пользователя

Независимо от способа, при блокировке приложение отключает пользователя от OpenVPN-сервера и блокирует сертификат.

Особенности блокировки сертификатов в OpenVPN

После блокировки ключа пользователя потребуется 1 час прежде чем OpenVPN проверит список заблокированных сертификатов. Это является особенностью реализации OpenVPN.

Ускорить этот процесс можно только перезапуском службы OpenVPN.

Блокировка через список активных подключений

В списке подключённых пользователей откройте меню действий и нажмите "Заблокировать":

Блокировка через страницу пользователя

Зайдите на вкладку "Пользователи", затем выберите пользователя и в контекстном меню "Редактировать".

В левом нижнем углу экрана нажмите "Заблокировать":
